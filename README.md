# RDR2 Game Clock 🤠

Веб-сайт для отслеживания игрового времени на сервере Red Dead Redemption 2.

## Как это работает

- **Соотношение времени**: 12 игровых часов = 1 реальный час
- **Множитель скорости**: 1 реальная секунда = 12 игровых секунд
- **Автоматическое отслеживание**: Игровое время автоматически рассчитывается на основе реального времени

## Как использовать

1. Откройте `index.html` в браузере
2. Введите текущее время на сервере в поле "Времени на сервере"
3. Нажмите кнопку "Синхронизировать"
4. Сайт будет отслеживать и показывать:
   - **Текущее время в игре** - время, которое сейчас на сервере
   - **Реальное время** - ваше местное время
   - **Прошло в реале** - сколько реального времени прошло с момента синхронизации
   - **Прошло в игре** - сколько игрового времени прошло с момента синхронизации

## Особенности

✅ Нет необходимости в интернете после загрузки  
✅ Данные сохраняются в браузере (localStorage)  
✅ Синхронизация сохраняется при закрытии и повторном открытии браузера  
✅ Адаптивный дизайн (работает на мобильных устройствах)  
✅ Тематическое оформление в стиле RDR2  

## Элементы интерфейса

- **Поле ввода времени**: Введите время сервера в формате HH:MM
- **Кнопка Синхронизировать**: Запускает отслеживание игрового времени
- **Кнопка Сброс**: Очищает все данные и начинает заново
- **Информационная панель**: Показывает время последней синхронизации

## Технические детали

- HTML5
- CSS3 с градиентами и анимациями
- Ванильный JavaScript (без зависимостей)
